# Pixel War Client


Clone this repo and run `npm run dev` in the cloned repo to use it! Make sure to install all needed dependencies, and modify the `.env` file with your ip.\
You also need to download and run the server to make it work.

## How to play

Connect to the ip of the client host with port 5173 to try the pixel war.\
Authentification is required to access the web app. You can either register or login very intuitively.
Once logged-in, just pick the color you want to use and click on the area of the canvas where you want your pixel to be placed.

\
Have fun!
